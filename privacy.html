Privacy Policy

Eileena Content Intelligence
Last updated: March 2026

1. Overview

This application is a private analytics tool operated by Andre Angkasa for the sole use of Eileena Angkasa.

2. Data Collected

Instagram post performance metrics (reach, impressions, saves, shares, comments, likes) via the Instagram Graph API. YouTube video performance metrics (views, watch time, retention) via the YouTube Data API and YouTube Analytics API.

3. Data Usage

Data is used exclusively to generate weekly content performance reports for the account owner.

4. Data Storage

All data is stored on a private home server (Mac mini) and in a private Google Sheet. No cloud databases or third-party storage services are used.

5. Data Sharing

No data is shared with, sold to, or accessible by any third party. The application has no public-facing components.

6. Data Retention

Data is kept indefinitely for trend analysis. The account owner may request deletion at any time.

7. Access

Only the account owner and the application operator (family member) have access to the data.
